## Quality Notes

Live transcription introduces predictable errors:

- Technical acronyms mangled (e.g., "FBHAI" for "FBI", "AC3" vs "NC3")
- Speaker-name attributions drift
- Sentence fragments when multiple speakers overlap
- Proper nouns (instructor/student names) occasionally miscapitalized

The transcripts are preserved **as-delivered** for authenticity. Curated summaries are in [`../WEEKLY_SUMMARY.md`](../WEEKLY_SUMMARY.md).

## Privacy

- Transcripts contain only the instructor's name and occasional student first names.
- No personal details, grades, or private conversations are included.
- Instructor attribution is retained as the course is publicly advertised as taught by Dr. Maryam Ahmed.
