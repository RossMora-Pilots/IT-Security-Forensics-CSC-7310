## Investigation Phases

See the main write-up at [`../FINAL_PROJECT_FORENSIC_INVESTIGATION.md`](../FINAL_PROJECT_FORENSIC_INVESTIGATION.md).

1. **Intake & Custody** — scope, questions, chain of custody
2. **Image Verification** — hash check, working copy creation
3. **File System Analysis** — MFT, deleted files, recycle bin
4. **Registry Analysis** — user activity, USB history, ShellBags
5. **Event Log Analysis** — logons, process creation, log clearing
6. **Timeline Reconstruction** — cross-source correlation
7. **Expert Report** — facts → evidence → conclusions
